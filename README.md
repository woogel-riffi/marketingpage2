# marketingpage

Eine einfache "Hallo Welt" Webseite erstellt mit Next.js und Tailwind CSS.

## Lokale Entwicklung

Installiere die Abhängigkeiten:

```bash
npm install
```

Starte den Entwicklungsserver:

```bash
npm run dev
```

Öffne [http://localhost:3000](http://localhost:3000) im Browser.

## Deployment auf Vercel

1. Erstelle ein Konto auf [Vercel](https://vercel.com)
2. Verbinde dein GitHub Repository mit Vercel
3. Vercel erkennt automatisch Next.js und konfiguriert das Projekt
4. Klicke auf "Deploy"

Alternativ kannst du die Vercel CLI verwenden:

```bash
npm i -g vercel
vercel
```

Die Webseite wird automatisch mit einem HTTPS-Zertifikat und einer benutzerdefinierten Domain bereitgestellt.
